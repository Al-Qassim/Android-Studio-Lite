# Android Studio Lite

Native Android app IDE built with kotlin and jetpack compose called Android Studio Lite. it allows the user to edit and manage his android apps code base, build them to apks that he can run on his android device

# features for 0.1v
the app should run directly on your phone and achieve the following:
- Create a new android project with single activity with compose code
- navigate between project files.
- view projects
- use Gradle in cloud or in GitHub action or any alternative to build apk when click run button.
- The apk is downloaded on device and install apk screen appear so user can try the result of the code.
- User should be able to compile the code again on device after changing the code in Android Studio Lite, and see the update.

## extra features for later
- Integrate with Git (user can push pull commit from your IDE)
- integrate with AI assistant with api key
- Syntax highlighting.
